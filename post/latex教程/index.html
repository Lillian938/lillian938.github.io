latex提供了许多"控制符号"来得到诸如希腊字母,数学运算符, 关系运算符,箭头等无法用ASCII键盘直接输入的符号.

1、常用的符号和公式表示：

小写希腊字母, 如\alpha,\beta, \gamma, \pi;

大写希腊字母, 如\Gamma, \Pi, \Omega;

二元运算符, 如\times(乘号), \cdot(点乘), \div(除号), \pm(加减号), \circ(小圆圈)等;

关系运算符, 如\leq(小于等于), \geq(大于等于), \subset(包含于), \supset(包含), \in(属于);

否定关系运算符, 如\not=(不等于), \not<(不小于), \not\supset (不包含);

箭头, \leftarrow, \rightarrow, \longrightarrow, \uparrow, \mapto等;

其它符号, \nabla(Nabla算子), \angle(角), \infty(无穷), \forall(任意), \exists(存在), \prime(导数的撇).$%$(百分号)，_（下划线）

3、大小符号,求和号, 积分号怎么表达？

数学环境中, 还有一类特殊符号, 即可变大小符号, 如求和号, 积分号等会因为符号后面内容的大小自动调整自己的大小: \sum（求和符号）\int（积分号）、\prod, \coprod, \oint, \bigcap, \bigcup, \bigsqcup, \bigvee, \bigwedge, \bigodot, \bigotimes, \bigoplus, \biguplus.例子:$\sum_{i=1}^n x_i=\int_0^\infty f(x)dx$

4、数学公式上下标的表示：

简单上标：上标借助符号是'^’，如下所示：a^2 + b^2 = c^2

复杂上标：可以将表达式放在{ }内，比如：a^{x^2+y^2}=c

简单下标：借助于’_‘就可以实现，比如：a_2 = b

复杂下标：可以将表达式放在{ }内，比如：a_{x^2+y^2} = d

总结：latex中，只要是复杂的表达式，一般都用花括号{}将其括起来

9、公式太长怎么换行？

当我们输入的公式较长时，最容易想到的方法是，在会出现越界的情况，使用强制换行\，但是这种方法在公式中行不通。通常，我们行间公式的换行可使用align 环境来实现。如

\begin{equation}
    F = ma;
\end{equation}

%等价的如下：
\[
    F = ma;
\]

% 换行公式，用align环境
\begin{align}
    &F = ma;  \\
    &y = ax;
\end{align}

%说明：上面“&”用于对齐，“\\”用于换行

%如果加“*”号，后面的标号不会在pdf中打印出来
\begin{equation*}
    F = ma;
\end{equation*}

%在换行环境下，如果只是想让某一行的标号不打印出来，只需要在换行“\\”之前加上“\notag”
\begin{align}
    &F = ma;  \notag \\
    &y = ax;
\end{align}
